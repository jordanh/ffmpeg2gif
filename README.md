# ffmpeg2gif
A shell script wrapper for ffmpeg simplifying conversion to animated gifs

## Requirements
   * [ffmpeg](https://ffmpeg.org/)

## Usage

```bash
$ ffmpeg2gif.sh -hv -f <framerate> -u <yscale> input.mov output.gif
```
